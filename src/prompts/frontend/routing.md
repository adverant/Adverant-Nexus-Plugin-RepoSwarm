# Routing Analysis

## Objective
Analyze the application routing structure and navigation patterns.

## Focus Areas

### 1. Route Structure
- URL hierarchy
- Dynamic routes
- Nested routes
- Route guards

### 2. Navigation
- Programmatic navigation
- Link components
- Navigation state
- History handling

### 3. Code Splitting
- Route-based splitting
- Lazy loading
- Prefetching strategies
- Loading indicators

### 4. SEO & Performance
- Static generation
- Server-side rendering
- Meta tag management
- Sitemap generation

### 5. Authentication Routes
- Protected routes
- Role-based access
- Redirect handling
- Session persistence

## Output Format
Provide route map visualization and navigation flow analysis.
