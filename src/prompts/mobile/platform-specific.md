# Platform-Specific Analysis

## Objective
Analyze platform-specific implementations in mobile applications.

## Focus Areas

### 1. Cross-Platform Strategy
- Code sharing percentage
- Platform abstraction layers
- Native module usage
- Platform-specific components

### 2. iOS Specifics
- Native integrations
- App Store guidelines compliance
- iOS design patterns
- Performance optimization

### 3. Android Specifics
- Native integrations
- Play Store guidelines
- Material Design adherence
- Android-specific optimizations

### 4. Device Features
- Camera/media handling
- Push notifications
- Background tasks
- Deep linking

### 5. Performance
- Startup time optimization
- Memory management
- Battery usage
- Network handling

## Output Format
Provide platform comparison matrix and optimization recommendations.
