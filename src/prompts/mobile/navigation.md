# Mobile Navigation Analysis

## Objective
Analyze navigation patterns and user flow in mobile applications.

## Focus Areas

### 1. Navigation Structure
- Stack navigation
- Tab navigation
- Drawer navigation
- Modal presentations

### 2. User Flow
- Authentication flows
- Onboarding sequences
- Deep linking support
- Back button handling

### 3. Performance
- Screen caching
- Lazy loading
- Transition animations
- Memory management

### 4. Accessibility
- Screen reader support
- Focus management
- Gesture alternatives
- Touch target sizes

### 5. State Persistence
- Navigation state
- Scroll position
- Form data
- Session handling

## Output Format
Provide navigation flow diagrams and UX recommendations.
