# Library Exports Analysis

## Objective
Analyze the export structure and module organization of the library.

## Focus Areas

### 1. Export Structure
- Named vs default exports
- Re-exports organization
- Barrel files
- Subpath exports

### 2. Module System
- CommonJS compatibility
- ESM support
- Dual package hazard
- Package.json exports

### 3. Entry Points
- Main entry point
- Submodule entries
- TypeScript declarations
- Source maps

### 4. Dependencies
- Peer dependencies
- Optional dependencies
- Dependency injection
- External bundling

### 5. Build Output
- Build targets
- Source maps
- Declaration files
- Changelog generation

## Output Format
Provide module dependency graph and export recommendations.
